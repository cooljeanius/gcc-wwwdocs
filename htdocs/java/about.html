 i These pages are maintained by the GCJ team, which consists of numerous volunteers and contributors. The web effort is coordinated by Anthony Green.

Please send feedback, problem reports and patches to our mailing lists.

Like the GCJ sources themselves, the web pages are under CVS control. Changes to the CVS repository immediately propagate to the web server and you can easily track any changes or contribute some of your own.

Just like software, we maintain a ChangeLog file for web page changes. Please help keep this up to date.
