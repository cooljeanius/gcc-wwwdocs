Cygnus Sourceware Project for the Javatm Language

We're developing a complete free software toolkit for developing in the Java Language.

New: The first part of the development environment which we are making available is gcj, we are releasing it on September 6, 1998. We have a tarball available now and it is being integrated into the EGCS compiler suite as an additional front end (it will show up in all future EGCS snapshots). Check out the gcj web page for more information.

Mailing lists

We have two mailing lists set up for our Java related work, java-announce and java-discuss. java-announce is a low-volume, moderated list which will have announcements related to this project. java-discuss is an open list where people are invited to discuss the software we're releasing, ask questions, post patches, etc. Both are administered by the ezmlm/idx mailing list software, or you can subscribe with the entry boxes below:

Papers describing our work

The engineers on this project have written a number of papers for conferences and magazines.

  • February 1999: "The Cygnus Native Interface for C++/Java Integration"
    [    HTML] [SGML/Docbook source]
    This is a programmer's manual which explains how to use a Java-aware C++ compiler (G++) to write native methods for Gcj.
    The CNI design and ideas were proposed in "Writing native Java methods in natural C++" (November 1997).

  • August 1998: "No Silver Bullet - Garbage Collection for Java in Embedded Systems".
    [HTML] [PS for presentation slides]
    Author: Alex Petit-Bianco.

  • November 1997: "Writing native Java methods in natural C++" discusses interaction between Java and C++. This is an early design document; see "The Cygnus Native Interface for C++/Java Integration" (February 1998) for the programmer's manual.
    [HTML] [SGML/Docbook source]
    Author: Per Bothner.

  • October 1997: "Compiling Java for Embedded Systems" was presented by Per Bothner at the Embedded Systems Conference West in San Jose. It provides a technical overview of our strategy and plans.

Other free Java-related projects

  • The Glass project is a different front end to gcc.
  • The Classpath project is writing a free replacement for the standard Java libraries.
  • Japhar is a free Java VM.
  • Kaffe is another free Java VM.
