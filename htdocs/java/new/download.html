Downloading GCJ

Current Release

The most current stable release is 2.95.1. The source distribution is provided in two parts: the compiler, gcj, and the runtime, libgcj.

Gcj is part of the Gnu Compiler Collection, GCC. Follow the links on http://gcc.cygnus.com/gcc-2.95/gcc-2.95.1.html to download the compiler. You will need either the full distribution, or each of the Core Compiler, C++ and Java distributions.

Libgcj 2.95.1 is available from ftp://sourceware.cygnus.com/pub/java/libgcj-2.95.1.tar.gz

Development Sources

The latest development sources are available in two forms: periodic snapshots, and directly from the cvs source repository.

Snapshots

GCC snapshots are made about once a week. They are available from ftp://gcc.cygnus.com/pub/egcs/snapshots/index.html.

Libgcj snapshots are made nightly, and are available from ftp://sourceware.cygnus.com/pub/java/snapshot.

cvs

The easiest way to get libgcj is to check it out from the anonymous cvs repository. Here's how: 
cvs -d:pserver:anoncvs@anoncvs.cygnus.com:/cvs/java login
  (password is ``anoncvs'')
cvs -z9 -d:pserver:anoncvs@anoncvs.cygnus.com:/cvs/java co libgcj
