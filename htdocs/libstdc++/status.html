Status

If you have downloaded GCC 3.0, STOP. You have the most recent libstdc++-v3 code available. All library snapshots are older than the code contained in GCC 3.0. We will continue to roll library snapshots on a pseudo-independant basis (at least, that's the plan). Once we have released another snapshot, superceding the version in GCC 3.0, this paragraph will be replaced with that information.

Compiling this library requires sources of a recent version of GCC. We suggest using CVS GCC. The latest library snapshot has also been tested with this recent GCC snapshot: 2001-05-14, which is available from a GCC mirror site under the snapshots directory as "2001-05-14/gcc[-g++]-20010514.tar.gz", or via CVS using the "gcc_ss_20010514" checkout tag.

The most recent version of this library that will build with GCC 2.95.2 is the ninth snapshot release, libstdc++-2.90.8.tar.gz.

If you are using GCC 2.95.3, then the most recent usable version of this library is still the ninth snapshot release, libstdc++-2.90.8.tar.gz, and you will need to apply this patch to mirror the changes made in gcc/glibc.

New things that work are in the latest release notes.

Known problems and bugs, which will force us to actually keep this file up-to-date, can be found in bugs.

Things that need to be done and are looking for volunteers can be found in: pick me.
