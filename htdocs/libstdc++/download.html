Getting the sources by FTP or CVS

Get the snapshot archive from

ftp://sourceware.cygnus.com/pub/libstdc++/

CVS repository available by anonymous CVS.

  • cvs -z9 -d :pserver:anoncvs@sourceware.cygnus.com:/cvs/libstdc++ login
  • enter ``anoncvs'' for the password
  • cvs -z9 -d :pserver:anoncvs@sourceware.cygnus.com:/cvs/libstdc++ co libstdc++
  • You only need to specify the repository root the first time; after that cvs will go to the correct place automatically. Using '-z9' is highly recommended, as it will reduce the bandwidth required.
  • You can browse the CVS repository over the web.

For those of you using Cygwin or Mingw32, you can avoid various problems building the library by using prebuilt binaries of snapshots from Mumit Khan's Cygwin pages.
