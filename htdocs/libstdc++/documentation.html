Introductory notes for libstdc++

This is a short list of text files pertaining to this implementation of ISO 14882. A brief description follows the name of the file.

  • BADNAMES - names to avoid because of potential collisions
  • BUGS
  • C++STYLE - coding style by example
  • CHECKLIST - a list of required features and their status.
  • COPYING - GPL v2 license terms
  • DESIGN - overview of the implementation plan
  • HEADER_POLICY - header naming and sub-include structure
  • PROBLEMS
  • README - directory structure
  • RELEASE-NOTES - instructions for building, using
  • TODO - tasks and known bugs
  • organization
  • Contributor checklist
  • Copyright assignment form for libstdc++-v3
