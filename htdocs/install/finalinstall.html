Final install wgcc

Now that wgcc has been built and tested, you can install it with `cd objdir; make install' for a native compiler or `cd objdir; make install LANGUAGES="c c++"' for a cross compiler (note installing cross compilers will be easier in the next release!).

That step completes the installation of wgcc; user level binaries can be found in prefix/bin where prefix is the value you specified with the --prefix to configure (or /usr/local by default).

If you don't mind, please quickly review the build status page. If your system is not listed, send a note to egcs@egcs.cygnus.com indicating that you successfully built and installed gcc. Include the output from running srcdir/config.guess. (Do not send us the config.guess file itself, just the output from running it!)

If you find a bug in gcc, please report it to egcs-bugs@egcs.cygnus.com.
