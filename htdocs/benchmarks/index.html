Benchmarking GCC

For evaluation of the code generated by GCC a number of benchmarks are available. Some people run these benchmarks on a regular basis and therefore allow to monitor how GCC's optimizations evolve over time and whether optimizations really make a difference on the tested benchmarks.

The GCC Benchmark suite

GCC has also it's own Benchmark suites which contains a number of smaller tests together with sources and results from these runs for different platforms.

SPEC 95

Diego Novillo has setup a Pentium III system that runs daily SPEC 95 tests. The results can be seen on the page http://www.cygnus.com/~dnovillo/SPEC/.

SPEC 2000

Similar to Diego's setup, Andreas Jaeger is running SPEC 2000 tests on AMD Athlon systems. For details check http://www.suse.de/~aj/SPEC/.
