GCC 3.0 Branch Status

This page contains information about the current status of the GCC 3.0 Branch.

At present, the GCC development tree is in a slush state, with the goal of stabilization for branching for GCC 3.0. See this message for more information about the current state of the tree, and the goals for the slush state.

Testing

Please follow these steps to test from the release branch:

  1. Obtain the gcc_build script. You can get it here.

  2. Run it as gcc_build checkout update test.

    See comments in the script itself if you are using read-only CVS access, or send mail to the GCC mailing list, if you have questions.

  3. Send the output to Mark Mitchell. Then it will be linked onto this page.

Thank you!
