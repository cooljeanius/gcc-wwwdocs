egcs project mailing lists

We have moved all mailing list functions from the Cygnus mailserver to the egcs machine. If you have problems sending mail to egcs.cygnus.com, note that we block spam sites and open mail relays by subscribing to MAPS RBL and ORBS. Please do not complain to us, but ask your server admin to fix his setup.

The conversion to egcs.cygnus.com has three key benefits:

  1. The delay between the time a message is posted to a list and the time it arrives at its final destination should be significantly reduced.
  2. We will be able to put in anti-spam filters on our lists without affecting Cygnus's mailserver (see above).
  3. More of the list functions will be automated.

Please read the subscribe/unsubscribe instructions carefully. They may have changed since you last visited this page.

  • egcs-announce is a read-only low volume list where we post announcements about releases or other important events.
  • egcs is a high volume list for general discussions about egcs. Anything not covered by other mailing lists is suitable for discussion on egcs.
  • egcs-bugs is a relatively high volume list for bug reports and general discussion about bugs in egcs.
  • egcs-patches is a relatively high volume list for patch submissions and discussion of particular patches. All patch submissions and all discussion for a particular patch should be sent to egcs-patches.
  • egcs-cvs is a read-only, relatively high volume list which tracks checkins to the egcs CVS repository.
  • egcs-testresults is a moderate volume list where test results for the egcs compilers are posted.

All the lists have a maximum message size. If your message exceeds more than 40kbytes, you should compress any attachments before sending it to the list.

When posting messages, please select an appropriate list for the message and try to avoid cross posting a message to several lists.

Web archives for all mailing lists are on-line; egcs, egcs-announce, and egcs-bugs, egcs-cvs, egcs-patches. egcs-testresults. The archives are searchable.

The search engine for the web archives is currently hosted at BitRange.com. If you have comments about the search function, send a copy to hp@bitrange.com.

To post a message, just send mail to listname@egcs.cygnus.com. Please refrain from sending messages in HTML, RTF or similiar formats.

To subscribe to a list, send mail to listname-subscribe@egcs.cygnus.com

To unsubscribe, send a mail note to listname-unsubscribe@egcs.cygnus.com

Each list also has a digest, which you can subscribe to by sending mail to listname-digest-subscribe@egcs.cygnus.com
