egcs project mailing lists

The egcs project has several mailing lists which may be of interest to users and developers. All the lists are under majordomo control with an open subscription and post policy.

  • egcs-announce is a read-only low volume list where we post announcements about releases or other important events.
  • egcs is a high volume list for general discussions about egcs. Anything not covered by other mailing lists is suitable for discussion on egcs.
  • egcs-bugs is a relatively high volume list for bug reports and general discussion about bugs in egcs.
  • egcs-patches is a relatively high volume list for patch submissions and discussion of particular patches. All patch submissions and all discussion for a particular patch should be sent to egcs-patches
  • egcs-cvs is read-only, relatively high volume list which tracks checkins to the egcs CVS repository.

To post a message, just send mail to listname@cygnus.com.

When posting messages, please select an appropriate list for the message and try to avoid cross posting a message to several lists.

All the lists have a 40 kilobyte maximum message size. If your message is more than 40kbytes, you might try compressing any attachments. If it is still more than 40kbytes, then you should put the attachments on a web or ftp server somewhere and note the location of the attachment in your message.

To subscribe to a list, send mail to listname-request@cygnus.com with a message body consisting of

subscribe listname your-email-address

To unsubscribe, send a mail note to the appropriate -request@cygnus.com address with a message body of unsubscribe listname your-email-address.

Web archives for all three mailing lists are on-line; egcs, egcs-announce, and egcs-bugs, egcs-cvs, egcs-patches.

Unfortunately the web archives are not searchable. We hope to replace Hypermail with another archiving engine with search capabilities in the future.
