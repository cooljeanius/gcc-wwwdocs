GCC mailing lists

If you have problems sending mail to the GCC mailing lists, note that we block spam sites and open mail relays by subscribing to MAPS RBL and ORBS. Please do not complain to us, but ask your server admin to fix his setup.

Please read the subscribe/unsubscribe instructions carefully. They may have changed since you last visited this page.

  • gcc-announce is a read-only low volume list where we post announcements about releases or other important events.
  • gcc is a high volume list for general development discussions about gcc. Anything not covered by other mailing lists is suitable for discussion on gcc.
  • gcc-help is a relatively high volume list for people searching for help in building gcc.
  • gcc-bugs is a relatively high volume list for bug reports and general discussion about bugs in gcc.
  • gcc-patches is a relatively high volume list for patch submissions and discussion of particular patches. All patch submissions and all discussion for a particular patch should be sent to gcc-patches.
  • gcc-cvs is a read-only, relatively high volume list which tracks checkins to the gcc CVS repository.
  • gcc-cvs-wwwdocs is a read-only, relatively low volume list which tracks checkins to the gcc webpages CVS repository.
  • gcc-testresults is a moderate volume list where test results for the gcc compilers are posted.

To post a message, just send mail to listname@gcc.gnu.org.

Policies

All the lists have a maximum message size. If your message exceeds more than 40kbytes, you should compress any attachments before sending it to the list.

When posting messages, please select an appropriate list for the message and try to avoid cross posting a message to several lists.

Please refrain from sending messages in HTML, RTF or similiar formats.

Archives

Web archives for all mailing lists are on-line (gcc-announce, gcc, gcc-bugs, gcc-help, gcc-patches, gcc-cvs, gcc-cvs-wwwdocs, and gcc-testresults), plus you can also download mbox formatted archives of each mailing list.

A search engine for the web archives is currently hosted at BitRange.com. If you have comments about the search function, send a copy to hp@bitrange.com.

Subscription

To subscribe to a list, send a blank mail to listname-subscribe@gcc.gnu.org

To unsubscribe, send a blank mail to listname-unsubscribe@gcc.gnu.org

Each list also has a digest, which you can subscribe to by sending mail to listname-digest-subscribe@gcc.gnu.org

Help

For further information on using the lists, send a blank mail to listname-help@gcc.gnu.org.

As a last resort, for example if you want to unsubscribe while your mail server is listed by ORBS, contact law@cygnus.com.
