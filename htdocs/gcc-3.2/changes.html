GCC 3.2 Changes, New Features, and Fixes

Caveats

  • This GCC release is based on the GCC 3.1 sourcebase, and thus has all the changes in the GCC 3.1 series. In addition, GCC 3.1 has a number of C++ ABI fixes in it which make its C++ compiler binary incompatible with the C++ compilers found in earlier GCC releases, including GCC 3.1 and GCC 3.1.1.
