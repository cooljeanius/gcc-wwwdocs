egcs-1.1 test strategy

  • No testsuite regressions on major platforms.
  • libg++ addon builds and passes on same platforms as we do testsuite regressions on.
  • Builds glibc and passes glibc testsuite on major supported platforms (x86, alpha, sparc). Testing on powerpc and m68k would also be appreciated.
    We are testing a development version of glibc, so you must check out a readonly copy of glibc from the cvs repository on egcs.cygnus.com.
    The glibc cvs repository is at egcs.cygnus.com:/cvs/glibc. It is readable via anonymous cvs in a manner similar to the egcs sources.
    You will want to check out the sources with the tag "glibc-4-egcs-1_1" to ensure that you're checking the right version of glibc.
  • Linux kernel builds and runs (and 2.1.??? TBD). Platforms will be x86, alpha. Sparc & PowerPC kernel tests would also be greatly appreciated.
  • Red Hat 5.1 builds without errors on x86, Alpha. Again, Sparc and PowerPC tests would be greatly appreciated.
  • RTEMS builds on the following RTEMS targets:
    • hppa1.1
    • i386, i386-elf, i386-go32
    • i960
    • m68k
    • mips64orion
    • powerpc
    • sh, sh-elf
    • sparc
