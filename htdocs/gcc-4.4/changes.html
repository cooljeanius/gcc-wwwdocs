GCC 4.4 Release Series
Changes, New Features, and Fixes

Caveats

  • __builtin_stdarg_start has been completely removed from GCC. Support for <varargs.h> had been deprecated since GCC 4.0. Use __builtin_va_start as a replacement.
  • Some of the errors of the C++ front end that may be downgraded to warnings by using -fpermissive are now warnings by default. They can be converted into errors by using -pedantic-errors.
