Anonymous read-only CVS access

In an ongoing effort to accelerate development of egcs and provide an open development environment, we are making the egcs CVS source repository available readonly to the public at large.

You'll be able to pick up the latest code whenever you want instead of waiting for periodic snapshots. You'll even be able to check out the release branch(es) since they also live in the CVS repository.

Also note that you may wish to review the CVS manual which is provided in texinfo form in the CVS sources.

  • Browse the CVS history online via your web browser.
  • Visit Cyclic Software to pick up a recent copy of CVS. Links to books on the subject can also be found there.
  • A link to a reasonably simple introduction to CVS can be found at Cyclic, which is also available in an HTML version

Using the CVS repository

Assuming you have CVS installed on your machine you can check out the egcs sources with the following sequence of commands:

Set CVSROOT in your environment to 

	:pserver:anoncvs@egcs.cygnus.com:/cvs/egcs

Or alternately add 

	-d :pserver:anoncvs@egcs.cygnus.com:/cvs/egcs
in the CVS commands below (place it immediately after "cvs" -- ie before any other CVS arguments). Issue the command "cvs login". You will be prompted for a password; reply with "anoncvs".

Issue the command 

	cvs -z 9 co egcs
to check out the egcs sources.

Once you've got the repository checked out "cvs update" will sync your local copy with the repository. See the CVS manual for additional information on how to use CVS.

Note

It may be necessary to obtain patches for common utilities when building from CVS. We recommend you read the FAQ entry regarding additional tools you may need. Another FAQ entry documents changes you will likely have to make to the gperf utility.

CVS tags and checkouts

You can check out the latest version of the egcs-1.1.x release branch with the following command: 

	cvs -z 9 co -regcs_1_1_branch egcs

By changing the "-r" argument above you can check out particular releases or snapshots or the latest snapshot.

  • egcs_1_1_2_release
  • egcs_1_1_1_release
  • egcs_1_1_release
  • egcs_ss_yymmdd
  • egcs_latest_snapshot

egcs project web pages

The web pages for the egcs project are also in the CVS repository and you can check them out, submit patches, etc just like you do for the compiler itself. Use "cvs co wwwdocs" to check out the web pages.

Enjoy!
