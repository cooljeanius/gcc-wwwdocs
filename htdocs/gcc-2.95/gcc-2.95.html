July XX, 1999

The GCC project is pleased to announce the release of gcc-2.95.

gcc-2.95 is the first release of the GNU Compiler Collection since the April 1999 GCC/EGCS reunification and includes nearly a year's worth of new development and bugfixes.

gcc-2.95 has been extensively regression tested and package tested. It should be reliable and suitable for widespread use.

gcc-2.95 has several new optimizations, new targets, new languages, and other new features. See the new features page for a complete list of new features found in gcc-2.95 releases.

The gcc-2.95 source release includes installation instructions in both HTML and plaintext forms in the install directory in the distribution. However, the most up to date installation instructions and build/test status are on the web pages. We will update those pages as new information becomes available.

The gcc project would like to thank the numerous people that have contributed new features, test results, bugfixes, etc. This amazing group of volunteers is what makes gcc successful.

And finally, we can't in good conscience fail to mention some caveats to using gcc-2.95.
