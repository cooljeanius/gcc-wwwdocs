The GCC project is pleased to announce the release of gcc-2.95. gcc-2.95 represents the first release of the GNU Compiler Collection since the April 1999 EGCS/GCC reunification.

gcc-2.95 is a major release of the GNU Compiler Collection and includes nearly a year of development and bugfixes. It has been extensively tested and is believed to be stable and suitable for widespread use.

See the new features page for a more complete list of new features found in gcc-2.95 releases.

The gcc-2.95 release includes installation instructions in both HTML and plaintext forms (see the install subdirectory in the gcc-2.95 distribution). However, we also keep the most up to date installation instructions and build/test status on our web page. We will update those pages as new information becomes available.

The gcc project would like to thank the numerous people that have contributed new features, test results, bugfixes, etc. This amazing group of volunteers is what makes gcc successful.

And finally, we can't in good conscience fail to mention some caveats to using gcc-2.95.
