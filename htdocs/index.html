egcs project home page

Welcome to the egcs project!

egcs is an experimental step in the development of GCC, the GNU C compiler. If you are new to the egcs project, you should probably read the egcs mission statement.

We want to work closely with contributors to help and encourage them to submit their changes for inclusion in egcs. The egcs project provides access to its development sources with weekly snapshots and anonymous CVS. All code continues to be assigned to the FSF exactly as before and is available to the gcc2 maintainers for ultimate inclusion into the gcc2 tree.

We will provide regular, high quality releases. We want those releases to work well on a variety of native (including GNU/Linux) and cross targets and use an extensive test suite as well as a benchmark suite to maintain and improve quality. egcs-1.1.2 is the current release. gcc-2.95 is the next major release.

egcs is pronounced "eggs", the c is silent.

Development snapshots and CVS Access

The egcs project makes weekly snapshots of its development sources and provides remote CVS access to its source tree.

The snapshots are available from ftp://egcs.cygnus.com/pub/egcs/snapshots/ and many mirror sites around the world.

Readonly CVS access is available for everyone on the internet, and read-write CVS access is available for major contributors.

Be aware that snapshots and CVS provides direct access to development sources, which can be unstable at times. If you are using the development sources, we recommend you sign up for one or more of the egcs mailing lists.
